# Book Store
An interactive book store with many titles you can choose from. 

## Technologies Used
- Frontend: Python
- Deployment: GitHub Pages

## Features
- interact with the books
- manage the book stores
- paying for books


## Installation
1. Clone the repository
2. Run `npm install`
3. Create `.env` file with API keys
4. Run `npm start`
